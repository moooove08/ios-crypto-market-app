Реалізований функціонал
Екран 1 — Список:

20 топ-монет з CoinGecko API
Іконка, назва, тікер, ціна USD, зміна 24г (зелений/червоний)
Пошук за назвою або тікером (локально)
Pull-to-refresh
Обробка помилок мережі з кнопкою «Повторити»
Екран 2 — Деталі:

Іконка, назва, ринкова капіталізація
High/Low 24г
Опис з API (або заглушка)
Кнопка «Додати в обране» зі збереженням у UserDefaults
Технічні рішення:

async/await для всіх мережевих запитів
Кешування зображень через NSCache + скасування завантажень при reuse
Dependency Injection через протоколи (CryptoRepositoryProtocol, APIClientProtocol)
ViewModels ізольовані від UI (@MainActor)
